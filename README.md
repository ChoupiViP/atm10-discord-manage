# ATM10 Discord Manager

Un bot Discord moderne permettant de gérer un serveur **Minecraft All The Mods 10: To the Sky** hébergé sous Docker.

## 🚀 Fonctionnalités

### Version v0.1

* ✅ Connexion à Discord
* ✅ Commandes Slash
* ✅ Commande `/ping`
* ✅ Architecture de projet prête à évoluer

## 🛠️ Technologies

* Python 3.12+
* discord.py
* Docker (prévu)
* RCON (prévu)
